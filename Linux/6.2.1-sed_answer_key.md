
# sed Practice Answer Key

Expected outputs after running the commands (without modifying files unless `-i` is used).

---
## ‚úÖ Easy Level

### 1) `sed 's/ERROR/ERR/' log.txt`
Lines beginning with `ERROR` change to `ERR ...` (first occurrence per line):
```
INFO 2025-09-12 08:10:01 Starting service
DEBUG 2025-09-12 08:10:02 Loaded config profile=default
ERR 2025-09-12 08:10:03 Connection failed for user alice
WARN 2025-09-12 08:10:04 Retrying in 5 seconds
ERR 2025-09-12 08:10:09 Timeout while contacting api.example.com
DEBUG 2025-09-12 08:10:10 Backoff set to 1000ms
INFO 2025-09-12 08:10:12 Service ready
ERR 2025-09-12 08:11:01 Disk space low: 5% remaining
CRITICAL 2025-09-12 08:11:05 System overheating
ERR 2025-09-12 08:11:06 DEBUG flag accidentally enabled
ERR 2025-09-12 08:12:00 User bob encountered error code E123
```

### 2) `sed 's/INFO/INFORMATION/g' log.txt`
All `INFO` tokens become `INFORMATION`:
```
INFORMATION 2025-09-12 08:10:01 Starting service
DEBUG 2025-09-12 08:10:02 Loaded config profile=default
ERROR 2025-09-12 08:10:03 Connection failed for user alice
WARN 2025-09-12 08:10:04 Retrying in 5 seconds
ERROR 2025-09-12 08:10:09 Timeout while contacting api.example.com
DEBUG 2025-09-12 08:10:10 Backoff set to 1000ms
INFORMATION 2025-09-12 08:10:12 Service ready
ERROR 2025-09-12 08:11:01 Disk space low: 5% remaining
CRITICAL 2025-09-12 08:11:05 System overheating
ERROR 2025-09-12 08:11:06 DEBUG flag accidentally enabled
ERROR 2025-09-12 08:12:00 User bob encountered error code E123
```

### 3) `sed 's/warning/WARN/Ig' system.log`
Case-insensitive replacements:
```
System boot at 2025-01-01 00:00:00
WARN: deprecated kernel parameter in cmdline
WARN: fan speed controller unavailable
Error: failed to mount /dev/sda1
error: fsck required on /dev/sda1
FAIL: service network.target
Success: applied udev rules
success: updated initramfs
```

### 4) `sed -n 's/failed/FAIL/p' report.txt`
Only lines where substitution occurred:
```
Line two: FAIL due to timeout
Line five: FAIL network check
```
