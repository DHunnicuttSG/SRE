# Here's a list of basic Linux commands

## Navigation and File System  

**pwd** (print working directory) Shows your current directory.  
**cd** (change directory) Changes the current directory.  
**cd** Go to your home directory.  
**cd ..**  Go up one directory.  
**cd /path/to/directory** Go to a specific directory.  
**ls**  Lists files and directories in the current directory.  
**ls -l** Long listing format (shows details like permissions, size, date).  
**ls -a** Show all files, including hidden files (starting with .).  
**ls -h** Human-readable file sizes.  
**ls -t** Sort by modification time.  
**mkdir** (make directory) Creates a new directory.  
**mkdir directory_name** Creates a directory named directory_name.  
**mkdir -p path/to/new/directory** Creates parent directories as needed.  
**rmdir (remove directory)** Removes an empty directory.  
**rm** Removes files or directories.  
**rm file_name** Removes a file.  
**rm -r directory_name** Removes a directory and its contents (use with caution!).  
**rm -rf directory_name** Forcefully removes a directory and its contents (use with extreme caution!).  
**touch** Creates an empty file or updates the timestamp of an existing file.  
**cp** Copies files and directories.  
**cp source_file destination_file** Copies a file.  
**cp -r source_directory destination_directory** Copies a directory and its contents.  
**mv** Moves or renames files and directories.  
**mv source_file destination_file** Renames a file.  
**mv source_file destination_directory** Moves a file to a directory.  

## File Content and Manipulation  

**cat name_of_file** (concatenate) Displays the contents of a file.  
**less** Displays the contents of a file one page at a time (use spacebar to go to the next page, q to quit).  
**head** Displays the first few lines of a file (default: 10 lines).  
**head -n 20 file_name** Displays the first 20 lines.  
**tail** Displays the last few lines of a file (default: 10 lines).  
**tail -f file_name** Follows the file as it grows (useful for log files).  
**echo** Displays text to the console screen.  
**wc** (word count) Counts lines, words, and bytes in a file.  
**grep** (global regular expression print) Searches for patterns in files.  
**grep "pattern" file_name** Searches for "pattern" in file_name.  
**grep -i "pattern" file_name** Case-insensitive search.  
**sed** (stream editor) Performs text transformations on files. (More advanced)  
**awk** Another powerful text processing tool. (More advanced)  

## System Information and Processes  

**uname** Displays system information.  
**hostname** Displays the hostname of the system.  
**date** Displays the current date and time.  
**cal** Displays a calendar.  
**who** Shows who is logged in.  
**ps** (process status) Lists running processes.  
**ps aux** Shows all processes in detail.  
**top** Displays a dynamic real-time view of running processes.  
**kill** Terminates a process.  
**kill process_id** Kills a process with the given ID.  
**df** (disk free) Shows disk space usage.  
**du** (disk usage) Shows disk usage of files and directories.  

## Permissions  

**chmod** (change mode) Changes file permissions.  
**chown** (change owner) Changes file ownership.  

## Networking  

**ping** Tests network connectivity to a host.  
**ifconfig or ip a** Shows network interface configuration.  
**netstat or ss** Shows network connections, routing tables, and interface statistics.  

## Other Useful Commands  

**man** (manual) Displays the manual page for a command.  
**man ls** Shows the manual page for the ls command.  
**clear** Clears the terminal screen.  
**history** Shows a history of previously executed commands.  
**exit** Closes the terminal.  
**history > history.txt** pipes the results of the command, history to the file history.txt.  
- use history to get a list of command then use !# to execute previous command wher # is the line number of the command.
